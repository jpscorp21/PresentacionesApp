Introducción

El proyecto consiste en una aplicación web de tipo SPA (de una sola página) que permite la gestión y la generación automática de presentaciones de tipo Google Slides con un formato compatible a su vez con Power Point. A su vez, la aplicación permite la exportación a formato PDF.

La aplicación web utiliza el api que provee Google para manejar tanto a Google Drive y a Google Slide, teniendo la posibilidad del aprovechamiento para crear el proyecto.

Mediante la carga de un texto en el formulario de la aplicación, esta permite posteriormente generar una presentación de manera profesional en función al tipo de tema seleccionado. Dicho formulario está construido utilizando el acrónimo WYSIWYG (What You See Is What You Get)

En el proyecto es necesario el uso de Internet y tener una cuenta de Google para su respectiva utilización. Cada presentación generada crea un pptx dentro de Google Drive, permitiendo su posterior manipulación.
