Generar una presentación

Para generar una presentación se debe haber ingresado con la cuenta de Google. Luego, dirigirse al menú principal y presionar el botón Nueva presentación. Una vez realizado se verá una pantalla como la siguiente:

Primeramente, todo el contenido que haya en el formulario grande de abajo se utilizará para la generación de la presentación. Para empezar es necesario definir el título y elegir un tema a gusto. Luego se debe borrar todo el contenido del formulario grande de abajo y escribir lo siguiente:

Las presentaciones tienen títulos para representar el contenido de manera global en qué consiste una diapositiva en concreta. Para definir un título se debe escribir el texto y luego hacer click en la sección "Normal" y elegir "Encabezado" para formatear el texto. Con esto la aplicación identifica el título.

Para identificar las diapositivas se separan en los "Títulos". Como ejemplo se muestra el siguiente texto cargado:

El siguiente texto está compuesto por un título y un texto. El texto debe estar con el formato "Normal" y el título con el formato "Encabezado". Este conjunto es otra diapositiva porque el título se encarga de la división de las diapositivas.

En la siguiente imagen se visualiza la carga de una imagen:

Para cargar una imagen se debe hacer click en el ícono que está al lado del boton "Normal" y aparecerá un menu para cargar la nueva imagen mediante la URL. Como ejemplo la correspondiente URL es la siguiente:

https://upload.wikimedia.org/wikipedia/commons/thumb/c/cf/Angular_full_color_logo.svg/250px-Angular_full_color_logo.svg.png

Una vez la imagen es cargada en el formulario, la aplicación se encargará de ubicar la imagen en la presentación final. Es posible agrupar un total de tres imágenes consecutivamente sin ningún texto.

Tener en cuenta lo siguiente, la lista no puede estar junto a un texto normal, es decir se elige lista o texto, pero si es posible añadir imagen junto a una lista.
